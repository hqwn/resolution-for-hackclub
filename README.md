# 🐍 Resolution for Hack Club — Python Pathway

My weekly projects from the [Hack Club Resolution](https://resolution.hackclub.com/app/pathway/python) Python pathway.

## About

[Resolution](https://resolution.hackclub.com) is a program by [Hack Club](https://hackclub.com) where participants pick a pathway and ship a project every week for 8 weeks. This repository contains all of the projects I built while following the **Python** pathway.

## Getting Started

Each week's project lives in its own folder. To run any project:

```bash
# Clone the repo
git clone https://github.com/hqwn/resolution-for-hackclub.git
cd resolution-for-hackclub

# Navigate to a week's folder and run it
cd week1
python main.py
```
